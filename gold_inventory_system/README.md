# نظام الجرد الشهري لمصنع الذهب

## المتطلبات
- Python 3.9+
- PySide6 أو PyQt5
- openpyxl

## التثبيت
```bash
python3 -m pip install -r requirements.txt
```

## التشغيل
```bash
python3 main.py
```

## المميزات
- إدارة الأصناف (إضافة/تعديل/حذف + بحث)
- إدخال الجرد الشهري مع التحقق ومنع القيم السالبة
- تقرير المقارنة بين شهرين مع تلوين الفروقات (+ أخضر / - أحمر)
- تصدير HTML و Excel + طباعة من المتصفح
- نسخ احتياطي واستعادة + تسجيل الأخطاء في logs/app.log

## ملاحظات
- الواجهة باللغة العربية وتدعم الاتجاه من اليمين إلى اليسار
- قاعدة البيانات SQLite مخزنة محلياً باسم gold_inventory.db
- يتم إنشاء نسخة احتياطية تلقائياً يومياً عند تشغيل التطبيق
